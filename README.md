# QZipper

## Prerequisites
install [libarchive](https://github.com/libarchive/libarchive)

```
$	tar xzf libarchive.tar.gz
$	cd libarchive
$	./configure
$	make
$	make check //opion
$	make install

```

copy and patse in to

```
src/library/QArchive/libarchive
```
